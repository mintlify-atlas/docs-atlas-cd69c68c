---
title: "DbBuilder"
description: "Advanced configuration builder for SlateDB databases"
---

The `DbBuilder` provides a fluent API for configuring and opening SlateDB databases with custom settings and components.

## Creating a Builder

### `Db::builder`

```rust
pub fn builder<P: Into<Path>>(
    path: P,
    object_store: Arc<dyn ObjectStore>,
) -> DbBuilder<P>
```

Creates a new database builder.

<ParamField path="path" type="P: Into<Path>" required>
  Path to the database in the object store
</ParamField>

<ParamField path="object_store" type="Arc<dyn ObjectStore>" required>
  Main object store for data persistence
</ParamField>

<ResponseField name="DbBuilder" type="DbBuilder<P>">
  A builder instance for configuring the database
</ResponseField>

## Configuration Methods

### `with_settings`

```rust
pub fn with_settings(self, settings: Settings) -> Self
```

Sets database settings (cache sizes, SST sizes, flush intervals, etc.).

<ParamField path="settings" type="Settings" required>
  Database settings to apply
</ParamField>

<ResponseField name="Self" type="DbBuilder<P>">
  Returns the builder for method chaining
</ResponseField>

**Example**

```rust
use slatedb::{Db, config::Settings};
use slatedb::object_store::memory::InMemory;
use std::sync::Arc;

#[tokio::main]
async fn main() -> Result<(), slatedb::Error> {
    let object_store = Arc::new(InMemory::new());
    let db = Db::builder("my_db", object_store)
        .with_settings(Settings {
            l0_sst_size_bytes: 64 * 1024 * 1024, // 64 MB L0 SSTs
            min_filter_keys: 2000,
            ..Default::default()
        })
        .build()
        .await?;
    Ok(())
}
```

### `with_sst_block_size`

```rust
pub fn with_sst_block_size(self, block_size: SstBlockSize) -> Self
```

Sets the block size for SSTable blocks.

<ParamField path="block_size" type="SstBlockSize" required>
  Block size variant (1KB, 2KB, 4KB, 8KB, 16KB, 32KB, or 64KB)
</ParamField>

<ResponseField name="Self" type="DbBuilder<P>">
  Returns the builder for method chaining
</ResponseField>

**Example**

```rust
use slatedb::{Db, config::SstBlockSize};
use slatedb::object_store::memory::InMemory;
use std::sync::Arc;

#[tokio::main]
async fn main() -> Result<(), slatedb::Error> {
    let object_store = Arc::new(InMemory::new());
    let db = Db::builder("my_db", object_store)
        .with_sst_block_size(SstBlockSize::Block8Kib) // 8 KiB blocks
        .build()
        .await?;
    Ok(())
}
```

**Note:** Blocks are the unit of reading and caching. Smaller blocks reduce read amplification but increase metadata overhead. Larger blocks are better for sequential scans but may waste bandwidth for point lookups. When compression is enabled, larger blocks typically achieve better compression ratios.

### `with_merge_operator`

```rust
pub fn with_merge_operator(self, merge_operator: MergeOperatorType) -> Self
```

Sets a merge operator for the database.

<ParamField path="merge_operator" type="MergeOperatorType" required>
  Arc-wrapped merge operator implementation
</ParamField>

<ResponseField name="Self" type="DbBuilder<P>">
  Returns the builder for method chaining
</ResponseField>

**Example**

```rust
use slatedb::{Db, MergeOperator, MergeOperatorError};
use slatedb::object_store::memory::InMemory;
use std::sync::Arc;
use bytes::Bytes;

struct CounterMergeOperator;

impl MergeOperator for CounterMergeOperator {
    fn merge(
        &self,
        _key: &Bytes,
        existing_value: Option<Bytes>,
        operand: Bytes,
    ) -> Result<Bytes, MergeOperatorError> {
        let existing = existing_value
            .map(|b| u64::from_le_bytes(b[..8].try_into().unwrap()))
            .unwrap_or(0);
        let delta = u64::from_le_bytes(operand[..8].try_into().unwrap());
        Ok(Bytes::from((existing + delta).to_le_bytes().to_vec()))
    }
}

#[tokio::main]
async fn main() -> Result<(), slatedb::Error> {
    let object_store = Arc::new(InMemory::new());
    let db = Db::builder("my_db", object_store)
        .with_merge_operator(Arc::new(CounterMergeOperator))
        .build()
        .await?;
    Ok(())
}
```

## Building the Database

### `build`

```rust
pub async fn build(self) -> Result<Db, Error>
```

Builds and opens the database with the configured settings.

<ResponseField name="Result" type="Result<Db, Error>">
  Returns the opened database instance or an error
</ResponseField>

**Example**

```rust
use slatedb::{Db, config::Settings};
use slatedb::object_store::memory::InMemory;
use std::sync::Arc;

#[tokio::main]
async fn main() -> Result<(), slatedb::Error> {
    let object_store = Arc::new(InMemory::new());
    let db = Db::builder("my_db", object_store)
        .with_settings(Settings {
            l0_sst_size_bytes: 32 * 1024 * 1024,
            ..Default::default()
        })
        .build()
        .await?;
    
    // Use the database
    db.put(b"key", b"value").await?;
    
    Ok(())
}
```
